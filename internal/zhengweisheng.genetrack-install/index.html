Genetrack-安装流程

丁桂涛, 郑炜圣, 朱乾书
Aug 27, 2020 2 分钟阅读时长 安装配置

#################################################

GeneTrack Ubuntu Linux x86_64安装 简易版 丁桂涛

################################################# 下载 GeneTrack-1.0.3_all.zip并解压
切换到解压之后到GeneTrack目录
安装python-virtualenv
sudo apt-get install python-virtualenv
virtualenv -p ./required/python2.5/bin/python –system-site-packages .
每次使用GeneTrack之前先输入
source PATH_TO_GENETRACK/bin/activate
export LD_LIBRARY_PATH=PATH_TO_GENETRACK/required/hdf5/lib(注：可以将这条加到~/.bashrc文件中，以后就不用每次输了)
测试GeneTrack

以上步骤在Ubuntu 12.04 x86_64上测试成功，如果这样安装不行，建议全新安装，参考下面安装步骤的完整版

#####################################################################

install Genetrack on Ubuntu 12.04 x86_64 GNU/Linux by Ding Guitao

#####################################################################

01. download Genetrack

cd ~/install/
wget http://genetrack.googlecode.com/files/GeneTrack-1.0.3.zip
unzip GeneTrack-1.0.3.zip

02. install sqlite3

wget http://www.sqlite.org/sqlite-amalgamation-3.6.20.tar.gz
tar zxvf sqlite-amalgamation-3.6.20.tar.gz 
cd sqlite-3.6.20/
./configure --prefix=/home/dgt/install/sqlite3
make
make install

03. install zlib-dev

wget http://fossies.org/unix/misc/zlib-1.2.6.tar.gz
tar zxvf zlib-1.2.6.tar.gz
cd zlib-1.2.6/
sudo make install

04. install python2.5

wget http://python.org/ftp/python/2.5.1/Python-2.5.1.tar.bz2
tar xjvf Python-2.5.1.tar.bz2
cd Python-2.5.1/
vim setup.py
    sqlite_inc_paths = [ '/usr/include',
                             '/usr/include/sqlite',
                             '/usr/include/sqlite3',
                             '/usr/local/include',
                             '/usr/local/include/sqlite',
                             '/usr/local/include/sqlite3',
                             '/home/dgt/install/sqlite-3.6.20/include',
                             '/home/dgt/install/sqlite-3.6.20/include/sqlite3',
                           ]
      disabled_module_list = ["bsddb185","dl","imageop"]
./configure --prefix=/home/dgt/install/python2.5 --with-zlib
make
make install

05. install python-virtualenv

sudo apt-get install python-virtualenv
virtualenv -p /home/dgt/install/python2.5/bin/python2.5 --system-site-packages /home/dgt/install/GeneTrack-1.0.3
source /home/dgt/install/GeneTrack-1.0.3/bin/activate

06. install SQLAlchemy

wget http://nchc.dl.sourceforge.net/project/sqlalchemy/sqlalchemy/0.7.6/SQLAlchemy-0.7.6.tar.gz
tar zxvf SQLAlchemy-0.7.6.tar.gz
cd SQLAlchemy-0.7.6/
python setup.py install

07. install Cython

wget http://cython.org/release/Cython-0.16.zip
unzip Cython-0.16.zip
cd Cython-0.16/
python setup.py install

08. install numpy

wget http://nchc.dl.sourceforge.net/project/numpy/NumPy/1.6.1/numpy-1.6.1.tar.gz
tar zxvf numpy-1.6.1.tar.gz
cd numpy-1.6.1/
python setup.py build
python setup.py install

09. install numexpr

wget 
cd numexpr-2.0.1/
python setup.py build
python setup.py install

10. install gc++ complier

sudo apt-get install build-essential

11. install hdf5

wget http://www.hdfgroup.org/ftp/HDF5/current/src/hdf5-1.8.8.tar.gz
tar zxvf hdf5-1.8.8.tar.gz
cd hdf5-1.8.8
./configure --prefix=/usr/local/hdf5
make
make check
make install
make check-install

12. install multiprocessing

wget http://pypi.python.org/packages/source/m/multiprocessing/multiprocessing-2.6.2.1.tar.gz
tar zxvf multiprocessing-2.6.2.1.tar.gz
cd multiprocessing-2.6.2.1/
python setup.py install

13. install pytables

wget http://nchc.dl.sourceforge.net/project/pytables/pytables/2.3.1/tables-2.3.1.tar.gz
tar zxvf tables-2.3.1.tar.gz
cd tables-2.3.1/
export HDF5_DIR=/usr/local/hdf5
export LD_LIBRARY_PATH=/usr/local/hdf5/lib
echo "export LD_LIBRARY_PATH=/usr/local/hdf5/lib" >>~/.bashrc
python setup.py build_ext --inplace
python
        >>> import tables
        >>> tables.test()
python setup.py install

14. install ChartDirector

wget http://download2.advsofteng.com/chartdir_python_linux_64.tar.gz
tar zxvf chartdir_python_linux_64.tar.gz
cd ChartDirector/lib/
cp -r * /home/dgt/install/python2.5/lib/python2.5/

15. GeneTrack test

python run.py
python run.py test
python run.py execute yeast.yeast_dataconf
python run.py server yeast.yeast_webconf
